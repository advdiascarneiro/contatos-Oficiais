# Página de Contatos — Dias Carneiro

Este repositório contém o código-fonte de uma **página estática de contatos**, desenvolvida em **HTML e CSS**, destinada à publicação via **GitHub Pages**.

A página tem como objetivo centralizar links públicos de contato e presença digital, como redes sociais, localização e canais externos já disponíveis ao público.

---

## Descrição Geral

- Página estática (sem backend)
- Não realiza coleta, armazenamento ou processamento de dados
- Não utiliza cookies
- Não possui formulários ou autenticação
- Conteúdo exclusivamente informativo

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- GitHub Pages (hospedagem)

---

## Segurança e Boas Práticas

Este projeto adota boas práticas básicas de segurança para aplicações estáticas, incluindo:

- Uso exclusivo de conteúdo local (`self`)
- Política de Segurança de Conteúdo (Content Security Policy – CSP)
- Links externos com proteção contra *tabnabbing*
- Ausência de scripts externos ou bibliotecas de terceiros

Por se tratar de um site estático e informativo, os riscos de segurança são mínimos.

---

## Publicação

O site é hospedado através do **GitHub Pages**, com HTTPS habilitado automaticamente pela plataforma.

---

## Observações Importantes

- Este repositório é **público**
- Não contém informações sensíveis, credenciais ou dados internos
- Todo o conteúdo exibido na página é de caráter público
- O código pode ser adaptado para fins educacionais ou de referência
